    ABSTRACT

    No presente trabalho foi necessária a criação de um ficheiro XSL através da utilização de um ficheiro XML contendo todos os filmes existentes até ao ano de 2018.

    Para tal, na criação do ficheiro XSL foram criados separadamente 3 índices, contendo os filmes, os géneros e os atores, respetivamente. Assim, todos os filmes (e os géneros e atores participantes respetivos) apresentam-se na página principal. De modo a visualizar cada filme, foi criada uma página individual para cada um desses filmes contendo os atores intervenientes nesse filme, bem como o(s) género(s) do filme.

    O mesmo foi feito para os géneros, onde foi criada uma página individual para cada um dos géneros, onde se apresentam todos os filmes que se inserem nesses géneros, bem como todos os atores desses filmes. Procedeu-se igualmente à criação de uma página individual para cada ator, onde se encontram os filmes em que esse ator encenou e, adicionalmente, os géneros desses mesmos filmes.

    De salientar que cada um dos filmes, géneros e atores tem um link associado, que nos leva consequentemente à página correspondente. Convém também realçar que todo o processo seria muito mais rápido com o recurso a uma "xsl:key", levando a sua não utilização a uma geração demorada do website dos filmes.
